Фруктовая таблица

Итоговый проект по предмету "Программирование".

Выполнила: ученица 10-го класса Власова Екатерина

Описание работы

Данная работа представляет собой интерактивную программу на языке Javascript, которая позволяет в легкой, игровой форме запоминать таблицу умножения. Дается пример и сверху падают фрукты - варианты ответов. Нужно выбрать правильный ответ. Если ученик отвечает правильно, скорость падения фруктов (сложность) постепенно увеличивается.

Запуск программы

Запустить программу можно, перейдя по этой ссылке.

ВНИМАНИЕ!

  • Программа не будет работать с локального диска. То есть она должна быть установлена на сайт (из-за ограничений безопасности языка Javascript)
  • Программа рассчитана на работу на настольном ПК или планшете с достаточно большим разрешением экрана
  • Работа программы проверялась в браузерах Firefox, Google Chrome, Opera

Использованные технологии

В данной работе применены такие технологии как: HTML и Javascript.

В игре задействован движок Phaser 3 для создания игрового пространства, спрайтов, анимации итд.
